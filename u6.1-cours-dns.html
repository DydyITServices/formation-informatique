Imaginez devoir retenir les coordonnées GPS exactes (Latitude: 48.8566, Longitude: 2.3522) pour aller chez chaque ami. Impossible, n'est-ce pas ? Vous utilisez leurs adresses postales.

En informatique, c'est pareil : les ordinateurs communiquent via des Adresses IP (le GPS), mais les humains préfèrent des Noms de Domaine (l'adresse postale).

2. L'Arborescence DNS

Le DNS est une base de données distribuée et hiérarchique. Il n'y a pas un seul serveur qui connaît tout Internet.

4. Les Enregistrements (Records)

Survolez les cartes pour voir la définition.

Type A

Standard IPv4

Associe un nom de domaine à une adresse IPv4 (ex: 192.168.1.1).

Type AAAA

Standard IPv6

Associe un nom de domaine à une adresse IPv6.

CNAME

Alias

Permet de créer un alias d'un nom vers un autre (ex: www vers domaine.com).

MX

Mail Exchange

Indique les serveurs de messagerie pour le domaine. Indispensable pour recevoir des emails.

PTR

Pointeur

Utilisé pour la résolution inverse (Trouver le nom à partir de l'IP).
